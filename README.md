# Microsoft Fabric Platform Enablement Program – HSØ

## Overview
This repository contains materials for the Microsoft Fabric Platform Enablement Program – HSØ, aimed at preparing healthcare organizations for Microsoft Fabric adoption. 

The program focuses on:

- Platform architecture
- Governance and capacity planning
- Data engineering workloads
- Analytics enablement
- Cross-organizational collaboration

## Structure
- `/slides` – Workshop presentations
- `/notebooks` – Demo notebooks and example transformations
- `/architecture` – Reference architecture diagrams
- `/governance` – Capacity and governance documentation
- `/exercises` – Optional hands-on materials
- `/docs` – Supporting notes and references
- `/datasets` – Sample datasets for demos and exercises
- `/pipelines` – Example data pipelines
- `/images` – Supporting images and diagrams
